About the Database

The database contains information about patents filed between 1980 and 2013. However, comprehensive information is only available from the year 2005. You can search for patent, inventor, lawyer, assignee, claims and citation information using this website.

Important Notes

  1. Not adding any filter will give you ALL the information we have on the field you requested! This means that if you select only the Patent ID, and submit that search with your email address, your resulting file will contain ALL the patent IDs we have! Please try to specify some filters as it greatly reduces time taken for the search and size of the resulting file.

  2. Dates should be entered in the ISO format: YYYY-MM-DD.

  3. If you provide a begin date and do not provide an end Date, the app will consider the end date as the current date. Similarly, if you do not provide a begin date, the app will oconsider the begin date as 1980-1-1.

Get Started

Click on "New Search" to search the the database. Click on "Check Status" to check the search queue and status of a previous search.
